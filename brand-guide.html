Glass Cards

Tres variantes: standard, gradient border, inner glow.

◈
Standard Glass Card
Background var(--glass) + border var(--glass-border) + backdrop-filter blur(12px). Hover: background muda, lift -3px.
◆
Gradient Border
Usa mask-composite para criar border gradiente. O ::before pseudo-element com -webkit-mask aparece no hover.
✦
Inner Glow
Radial gradient no ::after pseudo-element. Aparece no hover com fade suave, criando um glow interno accent.
/* Camadas do glassmorphism */
.glass-card {
  background: rgba(255, 255, 255, 0.03);       /* 1. Background translucido */
  border: 1px solid rgba(255, 255, 255, 0.07); /* 2. Border sutil */
  backdrop-filter: blur(12px);              /* 3. Blur do fundo */
  border-radius: 20px;
}
.glass-card:hover {
  background: rgba(255, 255, 255, 0.06);       /* 4. Hover state */
  transform: translateY(-3px);
}

04 — Efeitos

Efeitos & Interacoes

Animacoes e efeitos visuais que criam a experiencia premium.

Grain Overlay
SVG feTurbulence com 3.5% de opacidade. Fixed, full-screen, pointer-events: none. Adiciona textura organica ao background escuro.
Grain 3.5% opacity
Cursor Glow
Radial gradient 400x400px que segue o cursor. Accent color a 6% opacidade com blur natural. Apenas desktop — desactivado em touch devices. Esta pagina tem cursor glow activo.
Mova o cursor para ver
Fade-in Animation
IntersectionObserver trigger. translateY(40px) + opacity 0 para visible. Cubic-bezier(0.16, 1, 0.3, 1) — ease out expo. Delays em 0.1s increments para stagger.
Hover Lift
Cards elevam -3px a -6px no hover. Box-shadow aumenta. Transition 0.4s cubic-bezier.
Hover me
Hover me
Gradient Borders
Tecnica mask-composite: ::before com padding 1px, background gradient, mask linear-gradient content-box + xor.
Gradient border via mask-composite
Background Orbs
Radial gradients 400-600px, position fixed, blur(80px). Accent color a 3-5% opacidade. Animacao float 20-25s ease-in-out infinite. Criam profundidade no background.

/* JavaScript — IntersectionObserver */
const obs = new IntersectionObserver(
  entries => entries.forEach(e => {
    if (e.isIntersecting) {
      e.target.classList.add('visible');
      obs.unobserve(e.target);
    }
  }),
  { threshold: 0.15 }
);
document.querySelectorAll('.fade-in').forEach(el => obs.observe(el));
